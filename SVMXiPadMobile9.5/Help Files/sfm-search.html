Setup Requirements
This screen presents the SFM searches configured by your administrator.
Tap here to choose the criteria for the search string. Tap here to specify the search string. Tap here to set the number of records to be returned by the search. Swipe to turn off including search results from online. Tap here to run the specific search. Tap here to see search results for all sections. Tap on any object to see search results from that section only. Tap on any record to see all the configured fields. Tap here to download the record.
Rules
The layout, section, and fields for each search are fully configurable using SFM Search Designer.
You can use only those searches for which your administrator has given access.
Appropriate View processes should be available for viewing specific search result records.
SFM Search is available only from ServiceMax Summer 12 release.
Only master-level records can be downloaded.
Online results are indicated by a green bar. View processes cannot be launched for these.
Online Results will not be synchronized to the app. In order to download that data, modify your download criteria.
'Include Online Results' will be available only when there is connectivity.
