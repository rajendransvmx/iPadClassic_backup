Chatter is a collaboration application that helps you connect with coworkers inside your Salesforce.com organization and share business information securely and in real time. This screen allows you to view existing chatter posts related to the work order product, participate in discussion threads and to initiate new discussions. Tap on the numbers below for more information.
Tap to enter text using the keypad and start a new discussion thread. Tap on Share button when finished entering text. Tap to launch FaceTime for video calling. Tap to add your comments/inputs to an existing discussion thread. Tap on Comment button when finished entering text.
Rules
Chatter feature must be enabled in your Salesforce org in general, and for the product object specifically.
The image for the product is taken from the product record in Salesforce. Any changes to it can be done only online in Salesforce.
To learn about how to setup product pictures, please refer to the Setup Requirements.
User's FaceTime ID can be configured in the User record in Salesforce. If FaceTime ID is not setup, the user's Mobile phone and Email are used as FaceTime in the order of availability.
Your profile picture setup in Salesforce is not used here but should be added separately.
To learn about how to upload a picture to chatter, please refer to the Setup Requirements.
