Rules
This screen uses the time zone set on your iPad.
New events can be created only in Salesforce online.
All-day events, recurring events and multi-day events are not currently supported.
The calendar does not synchronize with iCal application.
Any changes made to work orders in iPad are updated to Salesforce immediately.
Calendar events are color-coded by priority and configured by ServiceMax administrators.
