Paramétrage requise
Cet écran montre les différentes options disponibles pour que vous permettre de créer de nouveaux enregistrements dans ServiceMax. Chaque option correspond à une transaction SFM configurée par votre administrateur. Les options sont classées par type d'enregistrement.
Appuyez sur l'étiquette de l'objet pour afficher les options Créer nouveau disponibles pour l'objet. Appuyez ici pour lancer l'écran Créer nouveau.
Règles
Votre administrateur doit vous permettre d'utiliser au moins une option Créer une nouvelle transaction.
Les enregistrements créés à partir d'ici peuvent être visualisés dans la section Récents de l'écran d'accueil.
Navigation
Vous ramène à l'écran d'information sur les ordres de travail. Tout changement apporté à l'information de débriefing n'est pas enregistré.
