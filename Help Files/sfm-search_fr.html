Paramétrage requise
Cet écran présente les recherches SFM configurées par votre administrateur.
Appuyez ici pour choisir les critères pour la chaîne de recherche. Appuyez ici pour spécifier la chaîne de recherche. Appuyez ici pour définir le nombre d'enregistrements à renvoyer lors de la recherche. Faites glisser pour désactiver, y compris les résultats de recherche en ligne. Appuyez ici pour exécuter la recherche spécifique. Appuyez ici pour voir les résultats de recherche pour toutes les sections. Appuyez sur un objet pour voir les résultats de recherche de cette section uniquement. Appuyez sur un enregistrement pour voir tous les champs configurés. Appuyez ici pour télécharger l'enregistrement.
Règles
La mise en page, la section et les champs de chaque recherche peuvent être entièrement configurés en utilisant Concepteur Recherche SFM.
Vous pouvez utiliser seulement les recherches pour lesquelles votre administrateur vous a autorisé l'accès.
Des traitements de visualisation appropriés doivent être disponibles pour visualiser les enregistrements de résultats de recherches spécifiques.
La fonction Recherche SFM ne sera disponible qu'à partir de l'édition Été 12 de ServiceMax. Cependant, la fonction « Télécharger à la demande » ne sera disponible qu'à partir de l'édition SP2 Été 12.
Les résultats en ligne sont indiqués par une barre verte. Les processus de visualisation ne peuvent pas être lancés pour ceux-ci.
« Inclure les résultats de ligne » sera disponible seulement lorsque vous êtes connecté.
Seuls les principaux enregistrements peuvent être téléchargés.
