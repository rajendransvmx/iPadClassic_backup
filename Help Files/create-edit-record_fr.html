Paramétrage requise
Cet écran présente un enregistrement Salesforce avec ses enregistrements dépendants, en mode Créer ou Éditer, de la manière configurée par votre administrateur en utilisant SFM Transaction Designer. En fonction du contexte du processus commercial, un même type d'enregistrement peut être édité en utilisant des écrans multiples, afin de capturer l'information dans des champs différents les uns des autres.
Appuyez ici pour afficher toutes les sections d'en-tête. Appuyez sur le nom de la section d'en-tête pour voir les informations de cette section uniquement. Appuyez sur le nom de la section dépendante pour ajouter/éditer des enregistrements pour la section. Appuyez sur le champ de recherche pour rechercher et sélectionner un enregistrement dans la fenêtre contextuelle. Appuyez sur le champ de liste de sélection multiple pour sélectionner une ou plusieurs valeurs dans la fenêtre contextuelle. Appuyez sur l'étiquette en pointillés pour afficher la totalité de l'étiquette. Appuyez ici pour rechercher et ajouter plusieurs enregistrements à la section dépendante. Appuyez ici pour ajouter un enregistrement sur la section dépendante. Faites glisser pour afficher le bouton SUPPRIMER et supprimer l'enregistrement. Appuyez ici pour afficher ou éditer l'enregistrement dépendant avec tous les champs configurés.
Règles
La mise en page, la section et les champs des écrans Créer/Éditer peuvent être entièrement configurés en utilisant SFM Transaction Designer.
Vous pouvez utiliser seulement les transactions SFM pour lesquelles votre administrateur vous a autorisé l'accès en utilisant l'assistant SFM.
Les enregistrements d'en-tête et de ligne sont enregistrés dans Salesforce seulement lorsque vous choisissez de les enregistrer. Les enregistrements de ligne ne font pas l'objet de sauvegardes intermédiaires.
Les valeurs de champs par défaut sont remplies de la manière configurée dans la transaction SFM.
Les transactions SFM personnalisées peuvent être uniquement lancées en mode Affichage.
Navigation
Affiche les actions classiques appropriées et tout événement de niveau de page personnalisé configuré par votre administrateur. Les actions classiques comprennent Enregistrement rapide, Enregistrer et Annuler.
Vous permet de capturer une signature d'approbation sur iPad et de l'enregistrer comme une pièce jointe avec cet enregistrement dans Salesforce. La signature est uniquement enregistrée lorsque vous sauvegardez l'enregistrement.
