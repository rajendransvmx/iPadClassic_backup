Setup Requirements
When creating new child records, you can use the Multi-add feature to find and add multiple records to a parent easily. This screen works similar to the standard lookup screen. However, you can select one or more records from the result set. Multi-add can be enabled on a child record if it contains any lookup fields. For example, multi-add can be used to add Parts records to Work Details.
Enter your search keyword here and tap the Search button on keyboard. Tap on Done to add the selected records. Tap on any selected row to deselect it. Tap on any unselected row to select it.
Rules
Multi-add must be configured by your administrator in SFM Transaction Designer.
Navigation
Closes the multi-add lookup window.
