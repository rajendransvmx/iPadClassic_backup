Rapport de service
Paramétrage requise
Cet écran affiche un résumé du travail réalisé avec toutes les lignes de débriefing. En utilisant cet écran, les clients peuvent rapidement examiner le coût de l'ordre de travail et l'approuver en signant sur les détails. Appuyez sur les nombres ci-dessous pour plus d'informations.
Appuyez pour lancer le panneau de capture de signature. Panneau de capture de la signature avec filigrane.. Appuyez sur le bouton Effectué après avoir capturé la signature. Appuyez sur le bouton Annuler pour annuler la capture de signature. Appuyez sur le bouton Effacer pour effacer le panneau de signature et entrer une nouvelle signature.
Règles
À chaque fois que l'information de débriefing est modifiée, la signature du client doit être de nouveau enregistrée.
Le rapport de service existant dans Salesforce sera écrasé.
Le rapport de service peut être uniquement généré à partir de cet écran.
Navigation
Génère le Rapport de service au format PDF et l'enregistre dans Salesforce.
Vous ramène à l'écran d'information sur les ordres de travail.
