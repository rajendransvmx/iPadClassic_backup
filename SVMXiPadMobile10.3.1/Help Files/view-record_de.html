Setup-Anforderungen
In diesem Fenster wird ein Datensatz von Salesforce mit seinen untergeordneten Datensätzen in einer vom Administrator im SFM-Transaktions-Designer konfigurierten Ansicht dargestellt. Es können mehrere Ansichten auf einen Datensatztyp zutreffen.
Tippen Sie hier, um in eine andere Ansicht des Arbeitsauftrags zu wechseln. Tippen Sie hier, um alle Kopfzeilen-Abschnitte anzuzeigen. Tippen Sie auf den Namen des Kopfzeilen-Abschnitts, um nur die Informationen zu diesem Abschnitt anzuzeigen. Tippen Sie auf den Namen des untergeordneten Abschnitts, um Informationen von untergeordneten Datensätzen anzuzeigen; dies kann auch darauf bezogene Objekte beinhalten. Zeigt den Synchronisierungsstatus an. Tippen Sie hier zweimal zur Aufgliederung und Anzeige der jeweiligen SFM-Ansicht. Tippen Sie auf die stilisierte Beschriftung, um die vollständige Beschriftung anzuzeigen. Tippen Sie auf die stilisierten Daten, um die vollständigen Daten anzuzeigen. Tippen Sie auf eine der SFW-Aktionen, die auf den Geschäftsprozess zutreffen. Tippen Sie hier, um mehr Details zur Produkt- und Kontohistorie herunterzuladen. Zeigt eine Konfliktmeldung an. Zeigt einen Zeitstempel der letzten Aktualisierung an.
Regeln
Layouts, Abschnitte und Felder in dieser Ansicht lassen sich vollständig mit dem SFM-Transaktions-Designer konfigurieren.
Sie können nur Ansichten nutzen, auf die Ihnen der Administrator Zugriff gewährt hat.
Spezielle benutzerdefinierte Optionen wie Fehlersuche, Serviceberichte und Chatter-Funktion können nur für Arbeitsaufträge aktiviert werden.
"Mehr anzeigen" funktioniert nur, wenn eine Verbindung besteht.
Navigation
Zeigt die vom Administrator konfigurierten benutzerdefinierten Aktionen, wie SFM-Transaktionen, an.
Führt Sie zum Fenster, in dem Sie den Datensatz zur Ansicht geöffnet haben.
