Paramétrage requise
Lors de la création de nouveaux enregistrements dépendants, vous pouvez utiliser la fonction Ajout Multiple pour trouver et ajouter de multiples enregistrements facilement à un enregistrement parent. Cet écran fonctionne de manière similaire à l'écran de recherche classique. Cependant, vous pouvez sélectionner un ou plusieurs enregistrements dans l'ensemble de résultats. L'ajout multiple peut être activé sur un enregistrement dépendant s'il contient des champs de recherche. Par exemple, l'ajout multiple peut être utilisé pour ajouter des enregistrements de pièces aux détails du travail.
Entrez votre mot-clé de recherche ici et appuyez sur le bouton Rechercher du clavier. Appuyez sur Effectué pour ajouter les enregistrements sélectionnés. Appuyez sur une ligne sélectionnée pour la désélectionner. Appuyez sur une ligne non sélectionnée pour la sélectionner.
Règles
L'ajout multiple doit être configuré par votre administrateur dans SFM Transaction Designer.
Navigation
Ferme la fenêtre de recherche d'ajout multiple.
