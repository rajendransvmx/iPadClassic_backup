Paramétrage requise
Cet écran présente un enregistrement Salesforce avec ses enregistrements dépendants, dans un affichage configuré par votre administrateur en utilisant SFM Transaction Designer. Il est possible d'appliquer plus d'un type d'affichage au même type d'enregistrement.
Appuyez ici pour passer à un affichage différent de l'ordre de travail. Appuyez ici pour afficher toutes les sections d'en-tête. Appuyez sur le nom de la section d'en-tête pour voir les informations de cette section uniquement. Appuyez sur le nom de la section dépendante pour voir les informations des enregistrements dépendants. Elle peut également contenir des objets associés. Affiche le statut de la synchronisation. Effectuez une double pression ici pour faire défiler et voir l'affichage SFM correspondant. Appuyez sur l'étiquette en pointillés pour afficher la totalité de l'étiquette. Appuyez sur les données en pointillés pour afficher la totalité des données. Appuyez sur une de ces actions SFW, applicables au processus commercial. Appuyez ici pour télécharger plus d'Historique sur le produit et d'Historique sur le compte, respectivement. Affiche le message de conflit. Affiche un horodatage de la dernière actualisation.
Règles
La mise en page, la section et les champs de cet affichage peuvent être entièrement configurés en utilisant SFM Transaction Designer.
Vous pouvez utiliser seulement les affichages pour lesquelles votre administrateur vous a autorisé l'accès.
Les options personnalisées spéciales telles que Dépannage, Rapport de service et Chatter peuvent être uniquement activées pour les ordres de travail.
« Voir plus » fonctionne uniquement si vous êtes connecté.
Navigation
Affiche les actions personnalisées telles que les transactions SFM configurées par votre administrateur.
