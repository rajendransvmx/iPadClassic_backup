In diesem Dokument werden Setup-Aktivitäten in ServiceMax/Salesforce ausgeführt, auf denen die Funktionalität und Benutzerfreundlichkeit des ServiceMax iPad-Clients basieren. Die Funktionen der iPad-Anwendung funktionieren nur dann wie vorgesehen, wenn das Setup wie im Folgenden ausgeführt abgeschlossen wird. Zum Durchführen der Schritte ist Erfahrung im Umgang mit dem Salesforce- und ServiceMax-Setup erforderlich.
Gerätevoraussetzungen
Der ServiceMax iPad-Client wird auf allen iPad-Modellen unterstützt. Die Mindestkonfiguration lautet iOS v4.2. Es ist eine Internetverbindung über 3G oder WLAN erforderlich. Wenn Ihr Netzwerk über einen Proxyserver oder eine Firewall verfügt, müssen Sie das iPad entsprechend konfigurieren, um auf salesforce.com zugreifen zu können.
Synchronisierung
Auf dem iPad-Client an Arbeitsaufträgen und sonstigen Objekten vorgenommene Änderungen werden sofort in Salesforce gespeichert und wieder mit der lokalen Datenbank synchronisiert, wenn eine Online-Verbindung besteht.
Datensatzsynchronisierung erfolgt in Einklang mit der mobilen Konfiguration in Salesforce.
Die Anwendung wird mit dem lokalen Datensatz offline weiterbetrieben, wenn keine Online-Verbindung besteht.
Produkt
Wenn Sie das Produktbild im Chatter-Fenster anzeigen möchten, erstellen Sie einen Bildanhang zum Produktdatensatz in Salesforce. Der Dateiname des Anhangs sollte das Wort "PICTURE" enthalten. Beispiel: CS2000 PICTURE.jpg oder PICTURE OF CS2000.jpg. Die folgenden Grafikformate werden unterstützt: jpeg, png, gif und bmp. Die Verwendung nicht unterstützter Bildformate kann zu unvorhersehbarem Verhalten in der Anwendung führen.
Erstellen Sie einen PDF-Anhang mit dem Wort "MANUAL" im Dateinamen, um Produkthandbücher im iPad-Client anzuzeigen. Beispiel: CS2000 MANUAL.pdf oder MANUAL für CS2000.pdf
Die Produkthandbücher werden heruntergeladen/zwischengespeichert, wenn eine Online-Verbindung besteht. Eine heruntergeladene/zwischengespeicherte Kopie wird bei nachfolgenden Aufrufen verwendet.
Chatter
Wenn Sie sich auf dem iPad mit Chatter austauschen möchten, aktivieren Sie Chatter in Salesforce und speziell für das Produktobjekt. Nähere Informationen zur Aktivierung von Chatter entnehmen Sie bitte der Salesforce Online-Hilfe.
Verfahren Sie in Salesforce wie folgt, um Ihr Foto zum Produkt-Chatter-Feed in der Anwendung hinzuzufügen.
  1. Erstellen Sie einen neuen Ordner im Bereich "Dokumente" mit der Bezeichnung ServiceMax für iPad. Wählen Sie die Einstellung Dieser Ordner ist für alle Benutzer zugänglich
  2. Erstellen Sie ein neues Dokument, dessen Name Ihrem vollqualifizierten Benutzernamen entspricht (z. B. ihrlogin@ihrunternehmen.com).
  3. Laden Sie Ihr Foto im PNG-Format hoch. Hinweis: Nur das PNG-Format wird unterstützt.
  4. Speichern Sie das Objekt im in Schritt 1 erstellten Ordner.
Kontakt
Erstellen Sie einen Bild-Anhang zum Kontaktdatensatz in Salesforce, um das Bild des Kontakts im Fenster mit den Auftragsdetails anzeigen zu können. Der Dateiname des Anhangs sollte das Wort "PICTURE" enthalten. Beispiel: John Doe PICTURE.jpg oder PICTURE OF John Doe.jpg. Die folgenden Grafikformate werden unterstützt: jpeg, png, gif und bmp. Die Verwendung nicht unterstützter Bildformate kann zu unvorhersehbarem Verhalten in der Anwendung führen.
Knowledgebase Artikel zur Fehlersuche
Verfahren Sie zur Erstellung von Knowledgebase-Artikeln wie folgt:
  1. Erstellen Sie einen Ordner, dessen Name genaumit dem Titel des Knowledgebase-Artikels übereinstimmt.
  2. Erstellen Sie eine Seite index.html , über die Hyperlinks, Grafiken und Videodateien in den Ordner eingebettet werden können. Bei den Hyperlinks kann es sich um relative lokale Pfade oder Web-URLs handeln.
  3. Komprimieren Sie den Ordner in einer Zip-Datei.
  4. Melden Sie sich in Salesforce an und navigieren Sie zur Registerkarte "Dokumente".
  5. Wenn noch kein spezieller Dokumentordner für Knowledgebase-Artikel angelegt wurde, erstellen Sie zunächst einen Ordner.
  6. Erstellen Sie wie folgt ein neues Dokument:
    1. Dokumentname = Titel des Knowledgebase-Artikels.
    2. Ordner = Name des in Schritt 5 erstellten Ordners.
    3. Beschreibung = Detaillierte Beschreibung dieser Lösung.
    4. Schlüsselwörter = Eine Liste mit durch Komma getrennten Schlüsselwörtern, die den genauen Produktnamen enthalten müssen, kann zum Auffinden des Dokuments verwendet werden. In der Regel Produktname, Familie, Art des Problems usw.
    5. Wählen Sie die in Schritt 3 erstellte Zip-Datei.
  7. Klicken Sie auf "Speichern".
Servicebericht
Verfahren Sie in Salesforce wie folgt, um das Logo auf dem Servicebericht zu ändern.
  1. Erstellen Sie wie folgt eine Grafikdatei mit dem Logo:
    1. Name = ServiceMax_iPad_CompanyLogo
    2. Format = nur PNG
    3. Maximum dimensions allowed = 600 (width) x 250 (height)
    4. Erforderliche Mindestauflösung = 72 dpi
  2. Laden Sie die Datei in den Ordner ServiceMax for IPad unter "Dokumente" hoch. Erstellen Sie den Ordner, sofern er noch nicht vorhanden ist.
