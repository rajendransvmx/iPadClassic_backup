Paramétrage requise
Cet écran affiche les événements programmés dans votre calendrier et les tâches qui vous sont assignées. Appuyez sur les nombres ci-dessous pour plus d'informations.
Appuyez sur la date requise pour voir le calendrier de cette date. Utilisez les flèches pour passer à un autre mois. Vous fournit une manière alternative de naviguer dans les jours du mois. Appuyez sur celui-ci pour passer directement au calendrier du jour. Vous permet d'afficher/ajouter des tâches à votre liste des choses à faire du jour. Vous permet de créer une nouvelle tâche. Dans la fenêtre contextuelle, entrez la description de la tâche, choisissez la priorité et appuyez sur Effectué. Affiche les événements programmés pour la journée. Touchez n'importe quel espace vierge dans la zone du calendrier, maintenez appuyé et faites défiler pour voir les événements hors de la plage horaire affichée. Appuyez sur un événement existant pour afficher l'enregistrement associé à l'événement. Appuyez sur un événement et faites-le glisser vers un horaire différent pour le reprogrammer. Appuyez sur le logo ServiceMax pour connaître l'identifiant d'utilisateur et la version de l'application que vous utilisez. Indique un conflit de synchronisation. Appuyez dessus pour voir les détails.
Règles
Cet écran utilise le fuseau horaire défini sur votre iPad.
Tout changement réalisé sur les événements dans iPad est immédiatement mis à jour dans Salesforce, si vous êtes connecté.
Le calendrier ne se synchronise pas avec l'application iCal.
Les événements d'une journée, les événements récurrents et les événements sur plusieurs jours ne sont pas encore pris en charge.
