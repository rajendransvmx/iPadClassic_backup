Setup-Anforderungen
In diesem Fenster werden die von Ihrem Administrator konfigurierten SFM-Suchvorgänge dargestellt.
Tippen Sie hier, um die Kriterien für den Suchbegriff auszuwählen. Tippen Sie hier, um den Suchbegriff anzugeben. Tippen Sie hier, um die Anzahl der Datensätze festzulegen, die von der Suche ausgegeben werden sollen. Streifen Sie zum Deaktivieren, einschließlich der Online-Suchergebnisse. Tippen Sie hier, um die spezifische Suche durchzuführen. Tippen Sie hier, um Suchergebnisse zu sämtlichen Abschnitten einzusehen. Tippen Sie auf ein beliebiges Objekt, um nur Suchergebnisse zu diesem Abschnitt anzuzeigen. Tippen Sie auf einen beliebigen Datensatz, um alle konfigurierten Felder einzusehen. Tippen Sie hier, um den Datensatz herunterzuladen.
Regeln
Layouts, Abschnitte und Felder für jeden Suchvorgang lassen sich vollständig mit dem SFM-Such-Designer konfigurieren.
Sie können nur Suchen verwenden, auf die Ihnen der Administrator Zugriff gewährt hat.
Für die Anzeige bestimmter Suchergebnisdatensätze sollten passende Ansichtsprozesse verfügbar sein.
Die SFM-Suche ist erst ab der Version "ServiceMax Summer 12" verfügbar. Die "Download On Demand"-Funktion ist jedoch erst ab der Version "Summer 12 SP2" verfügbar.
Online-Ergebnisse werden durch einen grünen Balken gekennzeichnet. Die Prozessansicht kann hierzu nicht gestartet werden.
Die Option "Beinhaltet die Ergebnisse der Online Suche" wird verfügbar, wenn eine Verbindung besteht.
Nur Master-Datensätze können heruntergeladen werden.
