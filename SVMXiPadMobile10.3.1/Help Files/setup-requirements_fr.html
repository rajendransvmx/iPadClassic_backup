Synchronisation
Ce document met en évidence les activités de paramétrage dans ServiceMax/Salesforce, qui supportent la fonctionnalité et l'expérience de l'utilisateur du client ServiceMax iPad. La fonctionnalité de l'application iPad fonctionnera comme prévu seulement si le paramétrage est réalisée comme indiqué ci-dessous. Il est nécessaire de se familiariser avec le paramétrage de Salesforce et de ServiceMax pour réaliser correctement les étapes.
Configuration matérielle requise
Le client ServiceMax iPad fonctionne avec tous les modèles d'iPad. La configuration minimale est iOS v4.2. Une connexion Internet est requise, soit par 3G soit par Wi-Fi. S'il existe un proxy ou un pare-feu sur votre réseau, l'iPad doit être configuré pour avoir un accès approprié à salesforce.com.
Synchronisation
Les mises à jour effectuées depuis le client iPad aux ordres de travail et autres objets sont immédiatement enregistrées dans Salesforce et synchronisées dans la base de données locale si vous êtes connecté.
La synchronisation des ensembles de données se produira conformément à la configuration mobile dans Salesforce.
L'application continuera à fonctionner hors ligne avec l'ensemble de données local si vous n'êtes pas connecté.
Produit
Pour afficher la photo du produit sur l'écran de Chatter, créez une pièce jointe image dans l'enregistrement du produit dans Salesforce. Le nom du fichier en pièce jointe doit contenir le mot PICTURE. Par exemple : CS2000 PICTURE.jpg ou PICTURE DE CS2000.jpg. Les formats d'images valides pris en charges sont jpeg, png, gif et bmp. L'utilisation de formats d'image non pris en charge peut entraîner un comportement imprévisible dans l'application.
Pour afficher les manuels des produits dans le client iPad, créez une pièce jointe PDF avec le mot MANUAL dans le nom du fichier. Par exemple : CS2000 MANUAL.pdf ou MANUAL pour CS2000.pdf
Les manuels des produits sont téléchargés/mis en cache lorsque vous êtes connecté. La copie téléchargée/en cache sera utilisée lors des invocations suivantes.
Chatter
Pour collaborer en utilisant la messagerie instantanée sur iPad, activez Chatter dans Salesforce et en particulier pour l'objet spécifique. Consultez l'aide en ligne de Salesforce pour plus d'informations pour activer Chatter.
Pour ajouter votre photo au fil de messagerie instantanée sur le produit dans l'application, effectuez les étapes suivantes dans Salesforce.
  1. Créez un nouveau dossier dans Documents appelé ServiceMax pour iPad. Choisissez l'option Ce dossier est accessible par tous les utilisateurs.
  2. Créez un nouveau document avec un nom correspondant à votre nom d'utilisateur complet (p. ex., votreidentifiant@votresociété.com).
  3. Chargez votre photo au format PNG. Veuillez noter que seul le format PNG est pris en charge.
  4. Enregistrez-le dans le dossier créé à l'étape 1.
Contact
Pour afficher la photo du contact sur l'écran de détails de tâche, créez une pièce jointe image dans l'enregistrement du contact dans Salesforce. Le nom du fichier en pièce jointe doit contenir le mot PICTURE. Par exemple : John Doe PICTURE.jpg ou PICTURE DE John Doe.jpg. Les formats d'images valides pris en charges sont jpeg, png, gif et bmp. L'utilisation de formats d'image non pris en charge peut entraîner un comportement imprévisible dans l'application.
Base de connaissances Articles pour le dépannage
Pour créer des articles dans la base de connaissances, effectuez les étapes suivantes :
  1. Créez un dossier dont le nom correspond
  2. exactement
    3. au titre de l'article de la base de connaissances.
  3. Créez une page index.html dans laquelle les liens hypertexte, les images et les fichiers vidéo peuvent être incorporés dans le dossier. Les liens hypertexte peuvent être des chemins d'accès relatifs en local ou des adresses URL du Web.
  4. Compresser le dossier.
  5. Connectez-vous à Salesforce et naviguez vers l'onglet Documents.
  6. Si un dossier de document dédié n'existe pas pour des articles de la base de connaissances, créez tout d'abord un dossier.
  7. Créez un nouveau document de la manière suivante :
    1. Nom du Document = titre de l'article de la base de connaissances.
    2. Dossier = nom du dossier créé à l'étape #5.
    3. Description = description détaillée de cette solution.
    4. Mots-clés = liste de mots-clés séparés par une virgule qui doivent contenir le nom exact du produit. Ils peuvent être utilisés pour localiser ce document. Typiquement le nom du produit, la famille de produit, le type de problème, etc.
    5. Choisissez le fichier Zip créé à l'étape #3.
  8. Cliquez sur Enregistrer.
Rapport de service
Pour modifier le logo sur le rapport de service, réalisez les étapes suivantes dans Salesforce.
  1. Créez un fichier image avec le logo de la manière suivante :
    1. Nom = ServiceMax_iPad_CompanyLogo
    2. Format = PNG uniquement
    3. Maximum dimensions allowed = 600 (width) x 250 (height)
    4. Résolution minimale requise = 72 dpi
  2. Chargez le fichier dans le dossier ServiceMax pour iPad sous Documents. Créez le dossier s'il n'existe pas encore.
