Setup Requirement
This screen allows you to enter details of the parts required, labor hours spent and expenses incurred while completing the work order. Once you have enterd all the debrief information, you can present the summary screen to the customer to obtain the approvel signature.
Feature Highlights
Parts
Allows you to view/change parts used in work order and add new parts to the work order.
Search
Shows the parts search screen using which you can add parts consumed. See Parts Serach for more information.
Labor
Allows you to view/edit labor hours for the work order. See Debrief Labor for more information.
Expenses
Allows you to view/edit expenses incurred for the work order. Refer yo Debrief Expenses for more information.
Rules
Parts prince is taken from ’Standard Prince Book’ in Salesforce
To delete an exisiting part from the work order set the quantity to zero.
Parts without a prince can not be added to the work order.
Parts inventory updates must be performed online in Salesforce.
Navigation
Shows work performed and debrif details such as parts, labor and expenses to facilitate approval through customer signature.
Save
Saves the debrief information to the work order in Salesforce and returns yot to work order details.
Back
Returns you to work order details screen. Any changes made to debrief information are not saved.
