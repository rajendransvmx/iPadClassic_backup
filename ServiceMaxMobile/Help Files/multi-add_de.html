Setup-Anforderungen
Beim Erstellen neuer untergeordneter Datensätze können Sie über die Option "Mehrere hinzufügen" ganz einfach mehrere Datensätze zu einem übergeordneten Eintrag suchen und hinzufügen. Das Fenster ähnelt in der Funktionsweise dem Standardsuchfenster. Sie können jedoch auch einen oder mehrere Datensätze aus den Ergebnissen auswählen. Die Option "Mehrere hinzufügen" kann für einen untergeordneten Datensatz aktiviert werden, wenn er Suchfelder enthält. Mit der Option "Mehrere hinzufügen" können Sie etwa Datensätze zu Teilen zu den Auftragsdetails hinzufügen.
Geben Sie hier Ihr Suchwort ein und tippen Sie auf die Suchschaltfläche auf der Tastatur. Tippen sie auf "Fertig", um die ausgewählten Datensätze hinzuzufügen. Tippen Sie auf eine beliebige markierte Zeile, um die Auswahl aufzuheben. Tippen Sie auf eine beliebige nicht markierte Zeile, um sie zu markieren.
Regeln
Die Option "Mehrere hinzufügen" muss vom Administrator im SFM-Transaktions-Designer konfiguriert werden.
Navigation
Schließt das Suchfenster mit der Option "Mehrere hinzufügen".
