Setup-Anforderungen
In diesem Fenster werden Ihre Optionen zur Erstellung neuer Datensätze in ServiceMax angezeigt. Jede Option entspricht einer von Ihrem Administrator konfigurierten SFM-Transaktion. Optionen werden nach Art des Datensatzes organisiert.
Tippen Sie auf die Objektbeschriftung, um die für das Objekt verfügbaren Neuerstellungsoptionen anzuzeigen. Tippen Sie hier, um das Fenster "Neu erstellen" aufzurufen.
Regeln
Ihr Administrator muss mindestens eine Neuerstellungstransaktion für Sie freigeschaltet haben.
Hier erstellte Datensätze können im Abschnitt "Neu" auf der Startseite angezeigt werden.
Navigation
Kehrt zum Fenster mit den Arbeitsauftragsdetails zurück. Any changes made to debrief information are not saved.
