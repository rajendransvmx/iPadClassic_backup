Setup Requirements
This screen shows various options available for you to create new records in ServiceMax. Each option corresponds to an SFM Transaction configured by your administrator. Options are organized by type of record.
Tap on the object label to view Create New options available for the object. Tap here to launch the Create New screen.
Rules
At least one Create New transaction must be enabled for your use by your administrator.
Records created from here can be viewed on the Recents section on Home screen.
Navigation
Returns you to work order details screen. Any changes made to debrief information are not saved.
